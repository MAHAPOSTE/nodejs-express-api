Description:
This is a simple Node.js server that performs operations on data.

Features:
- Get all data
- Find minimum and maximum values
- Calculate sum and average
- Handles HTTP requests

Tech Stack:
- Node.js
- HTTP module
- JavaScript

Project Structure:
- server.js → main server file

How to Run:
1. Clone the repository
2. Open terminal
3. Run:
   node server.js
4. Open browser:
   http://localhost:3000

API Endpoints:
- GET /all → returns all data
- GET /min → returns minimum value
- GET /max → returns maximum value
- GET /sum → returns sum
- GET /avg → returns average
